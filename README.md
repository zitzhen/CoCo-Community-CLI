# CoCo-Community-CLI
本工具作为[CoCo-Community](https://github.com/zitzhen/Coco-Community)的CLI，作为命令行交互。
> [!CAUTION]
>  我们已发布关于1-30-11-2025-12:55的[事故报告](https://github.com/zitzhen/CoCo-Community-CLI/tree/main/docs/accident%20report/1-30-11-2025-12%3A55)
# 安装&升级
## 安装
```bash
npm install -g @zitzhen/coco-cli
```
## 升级
```bash
npm update -g @zitzhen/coco-cli
```

## 命令指引
|名称|功能|
|---|---|
|ssl|为项目生成SSL证书|
|control|查询控件信息|
|search|搜索控件|
|clone|克隆项目|
|github|查看仓库URL与打开项目|
