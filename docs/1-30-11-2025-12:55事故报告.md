# 1-30-11-2025-12:55事故报告
时间： 2025年11月30日 12:55（UTC+8）

## 事件概述

2025年11月30日 12:55 起，CoCo-Community-CLI 开始受到影响。用户在更新 CLI 后，发现新功能无法正常使用。

## 排查结果

- 发布的版本号均正确

- 用户能够正常拉取更新

- 发布的源代码本身是正确的

## 问题原因：
在本地测试时，我们仅执行了：

```bash
tsc index.ts
node index.js
```

进行快速测试，并未执行完整的构建流程。因此发布到 npm 的包仍然包含 旧版本的构建文件，导致用户安装时无法获得最新功能。

## 已采取的措施

在 0.1.1 版本中，已重新执行完整构建流程并发布

修复了 CLI 包的入口及 dist 文件，确保用户能够正确获取更新

受影响的版本

- 0.0.2

- 0.0.3

- 0.1.0
