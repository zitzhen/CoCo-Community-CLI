## 事件概述

2025年11月30日 12:55 起，CoCo-Community-CLI 開始受到影響。用戶在更新 CLI 後，發現新功能無法正常使用。

## 排查結果

- 發布的版本號均正確

- 用戶能夠正常拉取更新

- 發布的源代碼本身是正確的

## 問題原因：
在本地測試時，我們僅執行了：

```bash
tsc index.ts
node index.js
```

進行快速測試，並未執行完整的構建流程。因此發佈到 npm 的包仍然包含 舊版本的構建文件，導致用戶安裝時無法獲得最新功能。

## 已採取的措施

在 0.1.1 版本中，已重新執行完整構建流程並發布

修復了 CLI 包的入口及 dist 文件，確保用戶能夠正確獲取更新

受影響的版本

- 0.0.2

- 0.0.3

- 0.1.0
